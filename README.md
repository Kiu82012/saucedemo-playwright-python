# SauceDemo Automation Project (Playwright + Python)

This is a simple End-to-End (E2E) automation project for the website [SauceDemo](https://www.saucedemo.com/).

## Features
- Automated Login (Standard User)
- Adding items to the shopping cart
- Complete Checkout process
- Automated Screenshots as test evidence
